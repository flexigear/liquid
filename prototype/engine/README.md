# Engine Workspace

The current minimal testable build lives in [prototype/engine/web](/D:/myWorkSpace/appProjects/liquid/prototype/engine/web).

This engine workspace is still intentionally lightweight. The browser prototype exists to validate:

- trackball-style container rotation
- square glass container readability
- droplet motion on inner faces
- merging behavior under gravity and angular inertia

If the project later moves to Unity, Godot, or a custom simulation stack, keep those heavier engine projects in their own subdirectory instead of replacing the current prototype history.
